# Project Memory

## Core
Sanjeevora NGO site. Brand: primary green #22C55E, accent sky blue #38BDF8, light tints #DFF6E3 / #E6F7FF, ivory bg.
Fonts: Plus Jakarta Sans (display, headings), Inter (body). Never serif.
Use gradient-brand / gradient-soft / text-gradient-brand utilities defined in styles.css for all brand gradients (NOT @utility — broken on this template).
Stack confirmed: TanStack Start + React 19 + Tailwind v4 + Lovable Cloud. User asked for Next.js 15 but accepted TanStack Start.
Donate is the primary CTA — keep it visible in header and as a quick-donate widget on the hero.
